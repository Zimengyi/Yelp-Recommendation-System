# 会议纪要 — 2026-05-01 (周五)

## Overview
ML2 期末 project 选题敲定：Travel-aware Restaurant Recommender。形态是一个 chatbot agent，用 LLM 做对话能力，背后调一个推荐系统当工具。重点分数项是推荐模型本身（DeepFM 路线），agent 和 UI 是产品包装。数据集走 Yelp Open Dataset，不需要 Google Maps API（Yelp 自带 lat/lon）。

## Features (v1)
**Feature 1: 主界面 — 基于地理位置的每日推荐**
进入 chatbot 默认展示。基于用户当前位置 + 当周热门，推荐附近餐厅卡片。卡片包含图片 / 时间建议 / 名字 / 评分 / 类别 / 一句话描述。

**Feature 2: Trip Plan — 行程感知三餐推荐**
chatbot 顶部按钮进入。用户给定多日行程，系统按"日 + 区域 + 时段"组织，每天 morning / lunch / dinner 各一家，区域内距离合理、跨日 cuisine 多样化。卡片同上 + Open route + Copy 按钮。
